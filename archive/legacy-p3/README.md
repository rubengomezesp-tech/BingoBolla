# Legacy P3 Archive

This folder preserves files moved out of the active BingoBolla source tree during the P3 repo hygiene pass.

## Contents

- `source-backups/`: historical `.bak` and `.backup` snapshots that used to sit inside `src/`.
- `install-scripts/`: local one-off shell scripts that generated or replaced world map and Ball Match files.
- `usa-puzzle-tour/`: an unrelated legacy React/FastAPI prototype, including its old test credentials note.

These files are not part of the active Next.js app and should not be used for new work unless intentionally recovering historical code.
